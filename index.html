Tall · entall/flertall · мій-моя-моє-мої · presens — og bokstaver Д Л Ч П
Tid: ca. 90 minutter Nivå: A1 Blokker: 4 · 18 oppgaver
Denne hjemmeleksen samler fire grunnsteiner: tall og hryvnia, entall og flertall med possessiver, verb i presens, og lese-trening for bokstavene Д · Л · Ч · П. Les teorien først, gjør så oppgavene i din egen takt og bruk fasit til å sjekke deg selv.

Grammatikk — kort og presist

1. Tall 1–10 og kjønnsformer

Tallene 1–10 på ukrainsk:

12345678910
один / одна / одне два / дві три чотири п'ять шість сім вісім дев'ять десять

1 og 2 endrer form etter substantivets kjønn:

  • один брат (m), одна сестра (f), одне місто (n)
  • два брати / два міста (m + n), дві сестри (f)

2. Pluss · minus · er lik

  • плюс = pluss · мінус = minus · дорівнює = er lik
  • Два плюс три дорівнює п'ять. — 2 + 3 = 5

3. Гривня — ukrainsk valuta

Tallet før гривня bestemmer formen:

TallFormEksempel
1гривняодна гривня

5–10гривеньдесять гривень

Det samme mønsteret gjelder копійка / копійки / копійок.

4. Entall og flertall (однина / множина)

De vanligste endringene fra entall til flertall:

MønsterEntallFlertall
konsonant → -ибрат, телефонбрати, телефони
-а → -икнига, сестракниги, сестри
-о → -амісто, вікноміста, вікна

5. Possessiver: мій / наш

Kjønnmin / mitt / minevår / vårt / våreEksempel
mмійнашмій брат, наш дім
fмоянашамоя сестра, наша сім'я

plмоїнашімої книги, наші батьки
Regel: possessivet retter seg etter ordet som kommer etter — ikke etter eieren.

6. Verb i presens

På ukrainsk viser endelsen på verbet hvem som handler. Personlig pronomen kan utelates.

PersonTypisk endelseжити (å bo)працювати (å jobbe)
я — jeg-у / -юживупрацюю
ти — du-еш / -єш / -ишживешпрацюєш
він/вона — han/hun-е / -є / -итьживепрацює
вони — de-уть / -ють / -ать / -ятьживутьпрацюють

Andre nyttige verb: писати (skrive), робити (gjøre), любити (like/elske), сидіти (sitte), стояти (stå), читати (lese), іти (gå nå), ходити (gå som vane).

7. Negasjon med не

Sett не rett foran verbet — verbet endres ikke.

  • Я працюю. → Я не працюю.
  • Вона живе тут. → Вона не живе тут.

Blokk D · Читання

16

Les stavelsene høyt

Прочитайте склади вголос
UTTALE

Les hver rad høyt — først sakte, så fortere. Читайте кожен рядок уголос — спочатку повільно, потім швидше.

  • Д: да · де · ди · до · ду · ді → день, дім, дитина
  • Л: ла · ле · ли · ло · лу · лі → лампа, любити, школа
  • Ч: ча · че · чи · чо · чу · чі → читати, чай, час
  • П: па · пе · пи · по · пу · пі → писати, привіт, парк
Tips: Ч er én sammensmeltet lyd som engelsk ch i chair — ikke del den opp i t+sj.

Les setningene høyt

Прочитайте речення вголос
LESING

Les hver setning to ganger — sakte, så naturlig. Прочитайте кожне речення двічі.

🔊 Setninger med Д · Л · Ч · П
Lytt først. Les så hver setning to ganger: sakte og naturlig.
  • Д · Добрий день! Це мій дім. Дідусь і дитина тут.
  • Д · У мене є друг, дочка і діти.
  • Л · Лікар любить читати. Люди стоять біля школи.
  • Л · Люба родина любить яблуко і молоко.
  • Ч · Чоловік читає. Ввечері він каже: «Добрий вечір!»
  • Ч · Чай, час і читання — маленька вправа для Ч.
  • П · Привіт! Це парк, а це подруга.
  • П · Петро пише приклади і повторює слова.
Tips: Gi ordene luft og rytme. Ikke dytt dem gjennom døra.
